1. Scala 简介

（1）变量和常量

var：变量

val：常量

尽量使用var，为了：

  1. 减少代码出错
  2. 使代码易读

Scala无需使用分号，编译器会自动判断语句的结尾。当一行中需要多条语句时，可以使用分号。

（2）条件语句

看起来语法和C语言类似，但是Scala的if语句会返回一个值，这个值是选择的分支的最后一行的值。

（3）方法（函数）

定义方法使用def关键词。在Scala中，形参的变量类型放在变量名后面，变量名和变量类型之间使用冒号进行分隔。在函数小括号的后面声明函数返回值的类型。

（5）递归函数和嵌套函数

一对大括号定义了一个代码区域，Scala的语法不支持在函数所在的代码区域外调用该函数。

上面注释的printRow(1)无法正常运行，因为调用在函数所在的代码区域外。

（6）列表

列表（Lists）能够用来表示集合和序列。列表和数组很像，但是有更多的操作函数，能够使用这些操作函数来添加和提取元素。

定义列表的三种方法：

  1. List函数生成法：val list1 = List(1, 2, 3)

    这一种办法将List看做了一个对象进行生成。

  2. 列举定义法：val list2 = x :: y :: y :: Nil

  3. 列表相加法：val list3 = list1 ++ list2

列表对象的方法：

  1. 获得长度：length
  2. 获得大小：size
  3. 获得头元素：head
  4. 获得后续元素（除了头元素的剩下的元素）：tail

List和数组一样都是以0为第一个索引，不过在索引的时候使用小括号而不是中括号。
