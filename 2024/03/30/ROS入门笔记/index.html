发表于 2024-03-30 更新于 2024-05-23 本文字数： 7.1k 阅读时长 ≈ 27 分钟

这个笔记是视频【鱼香ROS】动手学ROS2|ROS2基础入门到实践教程的笔记。

ROS是什么

ROS即Robot Operating System，机器人操作系统。这个“操作系统”实际上是软件库和工具集。

为什么用ROS

避免反复造轮子。

ROS和ROS2相比的缺点

  1. ROS不稳定
  2. ROS基于TCP通信，实时性差
  3. ROS对Python3兼容性差
  4. 没有加密机制，安全性不高

ROS2的升级

  1. 支持Windows、Mac、RTOS。
  2. 使用DDS进行通信。
  3. ROS2没有主节点。
  4. … …

功能包

ROS2中根据编译方式将功能包分为三种类型：

  1. ament_python
  2. cmake
  3. ament_cmake

获取功能包的方式：安装、手动编译

安装

通过安装获取功能包，功能包将被放置到/opt/ros/foxy/目录下。

launch文件介绍

为什么需要launch文件

因为需要启动的结点太多了，如果一行一行输入代码太慢了。

编写launch文件的格式

ROS2编写launch文件有三种格式：

  • python
  • xml
  • yaml

ROS2官方推荐python，因为python更灵活。
