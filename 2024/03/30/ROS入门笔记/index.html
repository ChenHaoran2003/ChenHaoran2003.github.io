发表于 2024-03-30 更新于 2024-04-02 本文字数： 2.1k 阅读时长 ≈ 8 分钟

这个笔记是视频【鱼香ROS】动手学ROS2|ROS2基础入门到实践教程的笔记。

ROS是什么

ROS即Robot Operating System，机器人操作系统。这个“操作系统”实际上是软件库和工具集。

为什么用ROS

避免反复造轮子。

ROS和ROS2相比的缺点

  1. ROS不稳定
  2. ROS基于TCP通信，实时性差
  3. ROS对Python3兼容性差
  4. 没有加密机制，安全性不高

ROS2的升级

  1. 支持Windows、Mac、RTOS。
  2. 使用DDS进行通信。
  3. ROS2没有主节点。
  4. … …

ROS2工作空间

一个工作空间能包含多个功能包，一个功能包能包含多个节点。

创建工作空间

创建工作空间的本质就是创建一个文件夹，不过工作空间的命名有着规范。

功能包

ROS2中根据编译方式将功能包分为三种类型：

  1. ament_python
  2. cmake
  3. ament_cmake

获取功能包的方式：安装、手动编译

安装

通过安装获取功能包，功能包将被放置到/opt/ros/foxy/目录下。

手动编译

通过apt-get install找不到可执行文件，或者想要修改源文件，可以手动编译。
